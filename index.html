2023-04-05
Cloudflare Pages & Workers

GPT and I decided to build an app for receipt splitting for group orders. The idea is to scan the receipt, generate a unique URL to email or SMS the group, and then allow each person to select their items from an itemized receipt. The app will also create a quick link to Venmo the owed amount to the person who paid the bill.

After some discussion, GPT and I chose to use Cloudflare Pages & Workers, partly because I wanted to try it. GPT created the home page, which is a form to add a scanned receipt, the tip amount, and the user's Venmo address.

The Cloudflare worker uses the Google Vision API for Optical Character Recognition (OCR), and GPT created parser logic to process the resulting OCR blocks into line items. Although GPT-4's image model might be able to do this more efficiently out of the box, I was astounded by how GPT-4 coded up a solution in seconds that only needed minor tweaks, like adjusting thresholds for items being on the same line.

We made amazing progress in a short amount of time, but the project is not finished yet.
